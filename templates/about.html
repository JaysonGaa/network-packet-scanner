About PacketScan

Educational network monitoring and security visualization tool

Project Overview

PacketScan captures and analyzes network packets using Python and visualizes traffic patterns through a Flask-based web dashboard. The project is designed to demonstrate network security concepts and monitoring techniques.

Technologies Used

  • Python (PyShark for packet capture)
  • Flask (Web framework)
  • Chart.js (Traffic visualization)
  • HTML / CSS (Frontend dashboard)

Possible Future Improvements

  • Charts and Graphs for Visualization
  • Database storage for historical packet logs
  • Live WebSocket updates
  • SIEM integration and alerting
