Common Network Vulnerabilities

Threats observable through packet-level monitoring

ARP Spoofing / Man-in-the-Middle

Attackers send forged ARP responses to redirect traffic through their system. Packet scanners can detect abnormal ARP replies and duplicate MAC-IP mappings.

Port Scanning

Reconnaissance technique where attackers send packets to multiple ports. Detected by identifying sequential connection attempts to many ports from one source IP.

DNS Tunneling

Malicious actors encode data inside DNS queries to bypass firewalls. Unusual domain lengths and high-frequency DNS requests can indicate tunneling.

Suspicious Protocol Usage

Unexpected protocols (e.g., ICMP floods or unusual TCP flags) may indicate malware command-and-control or denial-of-service activity.

Future Detection Features

  • Anomaly-based traffic detection
  • Machine learning classification of malicious flows
  • Real-time alerting system
